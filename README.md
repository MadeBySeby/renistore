# Renistore

Renistore is a modern, AI-enhanced e-commerce platform built with **Next.js 16**. It features a unique AI-driven price estimation tool, multi-language support, and secure authentication powered by Supabase.

Live Demo: [renistore.ge](https://renistore.ge)

## üöÄ Features

-   **AI Price Estimator**: Users can upload product images to instantly get a price estimation using AI (integrated with OpenAI & Hugging Face).
-   **Internationalization (i18n)**: Full multi-language support (English and Georgian) using `next-intl`.
-   **Secure Authentication**: User sign-up, login, and session management via Supabase Auth.
-   **Dynamic Product Catalog**: Real-time product fetching and display.
-   **User Dashboard**: Dedicated area for users to manage orders and view AI insights.
-   **Responsive Design**: Built with Tailwind CSS v4 for a seamless mobile and desktop experience.
